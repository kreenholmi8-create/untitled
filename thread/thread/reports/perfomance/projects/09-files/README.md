# Проект: Оптимизация I/O и сетевых операций в Java

## Описание
Проект демонстрирует сравнение трех подходов к работе с файлами в Java:
- RandomAccessFile
- FileChannel + ByteBuffer
- Memory-Mapped Files (MappedByteBuffer)

## Технологии
- Java 21
- Spring Boot 4.0.1
- Maven
- Lombok

## Запуск проекта

```bash
mvn clean install
mvn spring-boot:run
```
